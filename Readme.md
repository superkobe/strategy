## 项目介绍
- 本项目用于对投资策略的开源学习，项目包括数据的获取，策略的回测以及策略的开发
## 数据介绍
- 数据包括经济数据，基金数据以及股票数据，数据保存在mysql数据库中，在进行策略回测和开发时，可以从中进行读取
## 策略介绍
- 基金策略： 主要为指数基金的定投策略和分级基金的同级套利策略
- 股票策略： 对行业在不同维度上，提取相关因子进行二分类策略
